# LezuLogo

Shared Lezu wordmark component for the dashboard, marketing site, and docs.

## Usage

```vue
<template>
  <LezuLogo size="small" />
</template>

<script setup lang="ts">
import { LezuLogo } from '@lezu/ui'
</script>
```

## Props

- `size`: `'small' | 'medium' | 'large'`
- `animated`: `boolean`

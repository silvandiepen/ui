# TabBar

`TabBar` is a compact legacy tab-strip surface with a small API for switching between named views.

## When to use

- Use it where older product surfaces already depend on its compact variants.
- Prefer `Tabs` for newer work that needs pane composition, stronger slot support, or a more complete tab system.

## Notes

- `TabBar` is still useful for small dashboards and settings headers that only need a value-driven tab switcher.
- The docs app keeps it visible as transitional so replacement work does not get lost.

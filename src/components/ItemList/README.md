# ItemList

`ItemList.vue` renders a list of selectable items with optional row actions.

## API

- `items` (`ItemListItem[]`, default `[]`)
- `emptyText` (`string`, default `"No items found"`)

## Events

- `select(item)`
- `action({ item, action })`

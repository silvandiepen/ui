# Avatar

`Avatar` displays either a profile image or initials derived from a name.

## When to use

- Use it for people, organizations, or ownership metadata in lists and headers.
- Pass `src` for an image-based avatar or `name` to fall back to initials.

## Notes

- The current surface is intentionally small: `src`, `alt`, `name`, and `size`.
- Use surrounding components for status rings, menus, or profile detail.

# PlatformFooter

Shared shell footer used across products, with flexible slots for brand, navigation, meta content, and a secondary content row.

`colorMode` defaults to `auto`, which follows the document color mode through `data-color-mode` or `data-theme`, with `prefers-color-scheme` as a fallback. Pass `light` or `dark` to keep the footer fixed in that mode, or `inverse` to render light on dark pages and dark on light pages.
